1. GTM-модели: Что работает, что нет

FDE (Forward Deployed Engineering)

Инженеры вендора встраиваются в команду клиента. OpenAI: $350-630K comp. Рост вакансий +1,165%.

Palantir: $2.3B убытков до прибыльности. AIP bootcamps (75% конверсия) — реальный pivot.

Не для стартапов (ACV <$500K)

Consulting Alliances

OpenAI + Accenture/McKinsey/BCG. Accenture: $2.7B GenAI revenue, 77K AI pros.

Проблема: Non-exclusive. Accenture работает со всеми.

Для MWS: 1С-франчайзи, Softline, КРОК

Product-Led Growth

27% AI spend через PLG (4x vs traditional SaaS). Cursor: $0→$2B ARR без sales team.

Ограничение: Только dev-persona. Breaks при deals >$100K.

Работает для individual adoption

Bootcamp / Workshop

Palantir: 5 дней на реальных данных, 1,300+ bootcamps. OpenAI: 30-60 day pilots, ~100% win rate.

Для MWS: 2-3 часовой workshop — наиболее применимая модель.

Best fit для MWS

Agent-First (Skip Copilot)

Sierra ($100M ARR за 7Q), Harvey ($195M), Devin ($73M) — сразу агенты, без copilot-стадии.

Для MWS: HR Q&A = well-defined → agent-first оправдан.

Для well-defined workflows
Вердикт для MWS

FDE — преждевременно. PLG — как acquisition канал. Workshop на реальных данных клиента — основная GTM. Agent-first для HR Q&A. Партнерства с 1С-интеграторами для масштабирования.

Phase 1: Land (мес. 1-6)
HR KB agent-first + workshops на данных клиента. Per-seat pricing. Target: mid-market 500-5000 на Битрикс24/1С.

12. Legal / Compliance (152-ФЗ)

Что является ПД в HR-контексте

ФИО, паспорт, СНИЛС, ИНН, адрес, телефон, email, трудовая история, должность, зарплата, образование. Медданные (больничные) = СПЕЦИАЛЬНАЯ КАТЕГОРИЯ (ст. 10 — явное письменное согласие).

Штрафы (ред. 2024)

НарушениеПервый разПовторно
Отсутствие локализации ПД1-6M руб6-18M руб
Обработка без согласия300-700K рубдо 1.5M руб
Утечка (100K+ субъектов)до 500M руб+ уголовная

Кто отвечает за ошибки AI-бота

Работодатель (ТК РФ, ст. 22) — основная ответственность. Вендор — по договору. Специального закона об AI liability в РФ нет. Судебных прецедентов по AI HR-ботам нет.

Рекомендации для MWS
  • Disclaimer в каждом ответе: "справочный характер"
  • В договоре: AI = информирование, НЕ юр. консультация
  • Test suite на базе ТК РФ для валидации accuracy
  • Logging всех ответов для audit trail
  • НИКОГДА автоматические решения (увольнение, отказ)
